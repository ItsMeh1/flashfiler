Flash Filer

Professional Media & Application Runtime

What it is?

Flash Filer is a sandboxed browser engine designed for local media and secure code execution. Basically a mini-browser in your browser that runs on your browser.

How it works?

Select 'Import Document' to open files. The engine generates structured object environments instantly. Now you can run any modern & compatible file you wish.

Privacy

100% Secure. Everything runs on YOUR device only, and only you have access to any of the files you wish to view.

Navigation

When viewing a live asset, simply glide your cursor down to the lower boundary of the window to reveal controls.

The Flash Project

Flash Filer is part of The Flash Project, a project to bring ultimate freedom to the internets users.
